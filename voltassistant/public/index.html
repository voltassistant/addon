üìñ API Reference

Complete REST API for both local and cloud deployments

Health & Status

GET /health Health check endpoint
GET /api/status Full system status with HA connection info

Scheduler

GET /api/scheduler/status Current scheduler state and next run
POST /api/scheduler/start Start the scheduler
POST /api/scheduler/stop Stop the scheduler
POST /api/scheduler/tick Force immediate scheduler tick

Decisions

GET /api/decisions Decision history with filters

Configuration

GET /api/config Get current configuration
POST /api/config Update configuration

Home Assistant

GET /api/ha/verify Verify HA entity configuration
GET /api/ha/states Get current HA entity states

Cloud Sync

GET /api/cloud/status Cloud connection status
POST /api/cloud/link Link to cloud with API key

Authentication

POST /auth/register Create new account
curl -X POST http://cloud:3000/auth/register \ -H "Content-Type: application/json" \ -d '{"email":"user@example.com","password":"securepass"}'
POST /auth/login Login and get JWT tokens
POST /auth/refresh Refresh access token
GET /auth/me Get current user info

Installations

GET /installations List all your installations
POST /installations Create new installation
GET /installations/:id Get installation details
GET /installations/:id/decisions Get decision history

API Keys

GET /api-keys List your API keys
POST /api-keys Create new API key for addon linking
DELETE /api-keys/:id Revoke an API key

WebSocket

WS /ws?apiKey=... Real-time sync for addons
